# Egg-timer test requirements

The following links are to videos that set out the test requirements for [Parliament's egg-timer](https://api.parliament.uk/egg-timer/): 

[End-date test requirements](https://drive.google.com/file/d/12kL9x58mnmYTut3-PqwYDTW2gTsriZV9/view?usp=sharing)

[Start-date test requirements](https://drive.google.com/file/d/1yG_YBoch4tIojb8NTD47I9fopgqNu5SX/view?usp=drivesdk)

The links are viewable to anyone but the testing requirements can only be done by someone with access rights to the Google calendars. For more information please [contact](RIIDMSMailbox@parliament.uk) the Indexing and Data Management Section of the House of Commons Library.
